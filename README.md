# `~/.config/nvim`

Lua based config for nvim.

## Check nvim health
Open nvim and use the following command:
```
:checkhealth
```

## Dependencies
Python and node support for nvim. (node is optional)
- nvim python support
  ```
  pip install pynvim
  ```

- neovim node support
  ```
  npm -i g neovim
  ```
